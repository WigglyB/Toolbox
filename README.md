# Toolbox

Toolbox is a Java utility library containing reusable classes and systems that I commonly use across projects.

The goal of Toolbox is to provide lightweight, practical utilities without requiring large external dependencies.

## Features

### Expression System
A framework for representing and evaluating mathematical expressions.

Examples include:
- Constants
- Variables
- Addition
- Subtraction
- Multiplication
- Division
- Powers
- Roots
- Logarithms
- Summations

### File Utilities

#### ByteFile
A utility class for working with files as raw byte data.

#### Byteable
Interface for objects that can be converted to and reconstructed from bytes.

### Networking

#### Server
UDP server implementation.

#### Client
UDP client implementation.

#### PacketHandler
Utilities for sending and receiving packets.

### Utilities

#### Pointer<T>
A mutable reference wrapper.

### Ownership

#### Owner
Represents an owner.

#### Ownable
Interface for objects that have an owner.

## Example

```java
Map<String, Double> vars = new HashMap<>();
vars.put("x", 5.0);

Expression expr =
    new Add(
        new Variable("x"),
        new Constant(3)
    );

System.out.println(expr.evaluate(vars));
// Output: 8.0
```
